# ErrorCodes (표준 사전)
| 코드 | HTTP | 설명 |
|---|---|---|
| AUTH_REQUIRED | 401 | 인증 필요 |
| FORBIDDEN | 403 | 권한 부족 |
| VALIDATION_ERROR | 400 | 필드 검증 실패 |
| DUPLICATE_KEY | 409 | 고유키 충돌 |
| WO_STATUS_INVALID | 400 | 지시 상태 전이 불가 |
| CODE_INACTIVE | 400 | 비활성 코드 참조 |
| TIME_ORDER_INVALID | 400 | 시간 역전(또는 동일 시각 금지) |
| SERVER_ERROR | 500 | 내부 오류 |
